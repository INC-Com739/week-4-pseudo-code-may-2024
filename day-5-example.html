Sock Problem

Problem

Link to problem on HackerRank.

There are n colors of socks in a sock drawer. You want to find a pair of socks that are the same color. How many socks must you take out to be sure of having a pair of the same color?

Solution

The maximum number of socks you need to take out to be sure of having a pair of the same color is n+1.

Discussion

The solution to this problem is simple. The maximum number of socks you need to take out to be sure of having a pair of the same color is n+1. This is because if you take out n socks, you could have n-1 different colors of socks. The next sock you take out will be the same color as one of the socks you have already taken out.
